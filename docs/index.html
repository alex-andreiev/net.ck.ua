Current App: Book Reader

Book Reader is an offline Android reading app for importing, organizing, and reading books on device. It supports TXT, MD, FB2, EPUB, ZIP, and FB2.ZIP files and includes library management, reading progress, themes, and bookmarks.

The planned production version does not use login, ads, analytics, subscriptions, or cloud synchronization. The current project currently resolves its default application ID as com.example.coolreaderclone, which is still a placeholder and should be replaced before release.
