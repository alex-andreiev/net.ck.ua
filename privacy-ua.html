Політика конфіденційності Book Reader

Останнє оновлення: 19 червня 2026

Ця Політика конфіденційності описує, як Book Reader обробляє інформацію у зв'язку з роботою Android-застосунку.

Загальна інформація

Book Reader створений як офлайн-застосунок для читання. Для його роботи не потрібні реєстрація користувача, логін, рекламні ідентифікатори, підписки або хмарна синхронізація.

До яких даних застосунок може отримувати доступ

  • Файли книг і папки, які користувач вибирає для імпорту.
  • Метадані книг, витягнуті з імпортованих файлів.
  • Прогрес читання, закладки та налаштування читача, що зберігаються на пристрої.
  • Локальні crash-логи, що створюються на пристрої після збою застосунку.

Як використовується інформація

  • Для імпорту й відображення книг, обраних користувачем.
  • Для збереження позиції читання та персональних налаштувань.
  • Для організації локальної бібліотеки та покращення зручності.
  • Для діагностики технічних проблем за допомогою локальної інформації про збої.

Мережевий доступ

Запланована продакшн-версія Book Reader має працювати повністю офлайн. Вона не використовує аналітику, рекламні мережі або бекенд API для звичайної роботи читача.

Передача даних

Видавець навмисно не збирає книги, дані читання або персональні дані акаунтів на власні сервери через застосунок.

Зберігання даних

Дані, що зберігаються застосунком, зазвичай залишаються на пристрої користувача, доки їх не буде видалено користувачем або разом із застосунком.

Конфіденційність дітей

Застосунок не призначений спеціально для дітей, якщо інше прямо не зазначено на сторінці застосунку в магазині.
